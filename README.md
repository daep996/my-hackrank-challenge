## run challenges

### From JS (NodeJs v20)

```bash
node --watch challenge1/staircase.js
```

### From Python (v3.12)

```bash
python challenge1/staircase.py
```
### From Java (vX.y.z)

```bash
javac challenge1/Staircase.java
java challenge1/staircase.class
```
